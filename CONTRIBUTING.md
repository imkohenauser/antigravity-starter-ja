# Contributing to Antigravity Starter (Japanese Edition)

フィードバックや貢献を歓迎します！

## 行動規範

このプロジェクトに参加するすべての人に、敬意を持って協力的に振る舞うことを求めます。

## issue の報告

バグを発見した場合や機能要望がある場合は、Issue を作成してください。
提供されているテンプレート（Bug report / Feature request）を使用してください。

## Pull Request の送信

1. リポジトリを Fork してください
2. 機能用のブランチを作成してください (`git checkout -b feature/amazing-feature`)
3. 変更をコミットしてください (`git commit -m 'feat: Add some amazing feature'`)
4. ブランチに Push してください (`git push origin feature/amazing-feature`)
5. Pull Request を作成してください

### コミットメッセージの規約

このプロジェクトは [Conventional Commits](https://www.conventionalcommits.org/) に従っています。
`.agent/rules/git-commit-rules.md` を参照してください。

## 開発環境のセットアップ

```bash
npm install
```
